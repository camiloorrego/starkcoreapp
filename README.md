# StarkCoreApp

This project was generated with AngularJS version 1.7.8.

### Prerequisites

Node.js with npm

### Installation

``` bash
git clone https://github.com/camiloorrego/starkcoreapp.git
cd starkcoreapp
npm install
```

### Run

Run `npm start` for a dev server. Navigate to http://127.0.0.1:8000/index.html
